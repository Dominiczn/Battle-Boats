Objectives:
1. The system will display a menu on start up to the user to:
    - Start a new game
    - Resume a previous game (will only work if a previous game is saved)
    - Read the games instructions
    - Quit the game
  
2. The system will present the user with a blank 8x8 grid if the wish to play the game
  
3. The system will allow the user to select the location of 5 boats, making sure they are not overlapping:
   - 2 x Destroyers (1 cell)
   - 2 x Submarines (2 cells)
   - 1 x Carrier (3 cells)

4.  The system will randomly generate the locations of the computer's ships so that they are fully on the board and not overlapping

5.  The system will allow the user to select the target with a highlighted cursor rather than manual text input

6.  The system will display a blank target tracker
  
7.  The system will provide feedback to the user after each shot, saying whether it hit, missed, or sunk a ship on the target tracker (with H, M and S respectively)

8. The system will only allow the user to shoot at empty spaces

9.  The system will make the computer:
    - Randomly pick its target coordinate, making sure its an empty space
    - Target an adjacent coordinate if it hits a ship
  
10. The system will make the game continue until there is a winner that has sunk all the opponents ships and display it to the user. 

11. The system will save the games data to a file after each turn

12. The system will load the data from the save file if the user wants to carry on a previous game

13. The system will delete the previous games save data when the program is closed after a game is won
